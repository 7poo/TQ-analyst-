# Tuyên Quang score anomaly dashboard

Static analytics dashboard for reviewing unusual score sequences in the 2026 exam dataset.

## Vercel

Import this repository into Vercel and select **Other** as the framework preset. No build command or output-directory override is required.

The deployed site is generated from `index.html`, `app.js`, `data.js`, and the CSS files in the repository root.

> Statistical anomalies are screening signals for further review, not conclusions about misconduct or cause.
